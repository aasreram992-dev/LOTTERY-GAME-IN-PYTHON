# LOTTERY-GAME-IN-PYTHON
🎲 Lottery Game in Python

A simple lottery game built using Python and NumPy.
The program randomly generates a winning number between 0 and 99, and the user tries to guess it.

🚀 Features
Random winning number generation using NumPy
User input validation
Exception handling with try-except
Object-Oriented Programming (OOP) concepts
Beginner-friendly Python project
🛠️ Technologies Used
Python
NumPy
📚 Concepts Practiced
Classes and Objects
Methods
Loops
Conditional Statements
Error Handling
Random Number Generation
